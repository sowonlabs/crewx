# WBS Skill Test Project

This directory is used for testing the WBS (Work Breakdown Structure) skill functionality in CrewX.

## Purpose

The WBS skill provides structured project management capabilities for CrewX agents, enabling:

- Task breakdown and planning
- Job tracking and status management
- AI-based time estimation
- Phase-based project organization

## Usage

This test directory serves as a sandbox for validating WBS skill features during development.
