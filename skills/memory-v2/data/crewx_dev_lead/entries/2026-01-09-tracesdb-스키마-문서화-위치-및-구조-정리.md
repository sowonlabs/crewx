## 환경변수 연계
- CREWX_TASK_ID: 현재 Task ID (자식에 전달)
- CREWX_TRACE_ID: 분산 트레이스 ID
- CREWX_CALLER_AGENT_ID: 호출자 에이전트 ID
- CREWX_SKILL_SPAN_CREATED: Span 중복 방지 플래그

- crewx.tool.ts: Task 생성/완료
- skill.handler.ts: skill x 명령어 Span 생성
- skill.service.ts: skill run Span 생성
